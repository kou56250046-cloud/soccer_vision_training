瞬間認知 / 広さ
360°瞬間認知トレーニング
短時間だけ表示される複数ターゲットを、一度にどれだけ正確に覚えられるか。 ピッチ全体を「一枚の図」として捉える感覚を鍛えます。
指標：正答率 / 誤差 / 配置パターンごとの得意・不得意

予測 / 認知スピード
動きが途中で消える予測
直線・カーブ・ジグザグの軌道が途中で消えたあと、 数秒後のボール位置を予測。パスコースやこぼれ球の「先」を読む力を鍛えます。
指標：誤差px / 軌道タイプ別の精度 / 予測時間オフセット
開く ▶
周辺視 / ノールック
ノールック周辺視認知
中央の「＋」から目を離さない意識のまま、周辺に一瞬出るターゲットの位置を再現。 見ていない方向の味方・敵を感じ取る力を数値化します。
指標：平均誤差 / 命中率 / 左右・上下の正答率
開く ▶
スキャン頻度 / 反応
スキャン頻度トレーニング
一定時間内に、ランダムに現れるターゲットを何回キャッチできるか。 実戦に近い「首振りの回数/分」と反応時間の感覚を、指標として可視化します。
指標：回数/分 / 平均反応時間 / 視線の左右・上下バランス
